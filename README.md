## Setup

```sh
$ npm install -g bg-pdronenko-hexlet
```

## Run game

```sh
$ brain-even
```

```sh
$ brain-calc
```

```sh
$ brain-gcd
```

```sh
$ brain-progression
```

```sh
$ brain-prime
```
